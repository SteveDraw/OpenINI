# OpenINI
>:snowflake:相应.ini配置文件的读取写入保存等操作GUI窗口的类库项目，可引用编译输出的dll进行使用或者二次开发
## 1.有两类窗口，一个是原版window form风格的GUI，另外一个是使用sunnyUI封装的GUI窗口；
## 2.功能使用包括：
- :one:.ini文件读取解析，包括可选择不同类型的.ini文件读取；
- :two:.ini文件修改参数；
- :three:.ini文件修改参数；
- :four:.ini文件删除对应参数；
